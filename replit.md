# Dormy Landing Page

## Overview
A modern, fully responsive landing page for "Dormy" - a mobile application that digitizes dormitory processes for students. The page is designed to announce the upcoming app launch, showcase features, and collect email subscriptions.

## Target Audience
University students aged 16-24 living in dormitories.

## Tech Stack
- **HTML5** - Semantic markup
- **CSS3** - Mobile-first responsive design with CSS custom properties
- **JavaScript** - Vanilla JS for interactivity (no frameworks)
- **Vite** - Development server and build tool

## Color Palette
- **Primary**: #2d62a8 (Blue)
- **Background**: #eeedeb (Off-white)
- **Text**: #1a1a1a (Dark)

## Page Sections
1. **Header/Navigation** - Fixed header with navigation links
2. **Hero** - Main value proposition with phone mockup
3. **Features ("Ne Yapıyoruz?")** - 5 feature cards showcasing app capabilities
4. **Email Subscription ("Önce Senin Haberin Olsun")** - Form with anti-spam checkbox
5. **FAQ** - Accordion-style frequently asked questions
6. **About ("Biz Kimiz")** - Mission, vision, and team members
7. **Contact ("İletişim")** - Social links and contact form
8. **Footer** - Social media links and copyright

## Features Showcased
- Online Reservation System
- Smart Fault and Request Reporting
- Dormitory Sharing System
- FAQ Module
- Instant Notification System
